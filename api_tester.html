GET /webcam/state

GET http://localhost:8000/webcam/state
GET /stats
총 요청 수, 클래스별 카운트, 평균 신뢰도, 평균 레이턴시를 반환합니다. 피드백 루프 기준 지표.
GET http://localhost:8000/stats
GET /latest
아두이노 step8이 0.5초마다 폴링하는 엔드포인트. 최신 다중 감지 classes 배열 반환.
GET http://localhost:8000/latest
GET /collected
